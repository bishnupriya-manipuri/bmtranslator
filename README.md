# বিষ্ণুপ্রিয়া মণিপুরি ↔ বাংলা অনুবাদক

Bishnupriya Manipuri ↔ Bengali Translator — corpus-based, fully static website.

## ফাইল তালিকা

```
index.html              ← মূল ওয়েবসাইট
bm_bengali_corpus.csv   ← অনুবাদ corpus (দুই কলাম)
README.md               ← এই ফাইল
```

## GitHub Pages এ চালু করার নিয়ম

1. এই তিনটি ফাইল একটি GitHub repository তে আপলোড করুন
2. Repository Settings → Pages → Source: **main branch / root** সিলেক্ট করুন
3. কিছুক্ষণ পর সাইট চালু হয়ে যাবে

## নতুন শব্দ যোগ করার নিয়ম

`bm_bengali_corpus.csv` ফাইল খুলুন (Notepad বা Excel দিয়ে) এবং নিচে নতুন লাইন যোগ করুন:

```
বিষ্ণুপ্রিয়া শব্দ,বাংলা অনুবাদ
```

**উদাহরণ:**
```
মি,আমি
থাকাত,ধন্যবাদ
```

## বৈশিষ্ট্য

- বিষ্ণুপ্রিয়া → বাংলা অনুবাদ
- বাংলা → বিষ্ণুপ্রিয়া অনুবাদ
- শব্দ ব্রাউজ করার সুবিধা
- সম্পূর্ণ বিনামূল্যে (GitHub Pages)
- কোনো server বা backend দরকার নেই
